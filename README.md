# Superstore Sales & Profitability Analysis Dashboard (Advanced Excel)

## Project Overview
Developed an interactive Advanced Excel dashboard to analyze Superstore sales performance, profitability trends, customer segments, product performance, and discount impact using a dataset containing 9,995 transaction records.

This project focuses on transforming raw Superstore sales data into actionable business insights through data cleaning, pivot tables, calculated metrics, charts, and dashboard visualization.

## Dataset Information
The dataset contains the following fields:

- Row ID  
- Order ID  
- Order Date  
- Ship Date  
- Ship Mode  
- Customer ID  
- Customer Name  
- Segment  
- Country  
- City  
- State  
- Postal Code  
- Region  
- Product ID  
- Category  
- Sub-Category  
- Product Name  
- Sales  
- Quantity  
- Discount  
- Profit  

**Total Records:** 9,995 rows

## Project Workflow

### 1. Data Cleaning
- Converted raw dataset into Excel Table format  
- Checked missing values  
- Removed duplicate records  
- Corrected data types for dates and numeric columns  

### 2. Data Transformation
Created calculated columns:
- Profit Margin (%)  
- Delivery Days  
- Order Year  
- Order Month  

## Analysis Performed

### Sales and Profit by Category
Analyzed category-wise contribution to total revenue and profitability.

### Profit Margin Analysis
Compared category-level profitability efficiency.

### Segment Analysis
Evaluated sales performance across:
- Consumer  
- Corporate  
- Home Office  

### Sales Trend Analysis
Identified monthly and yearly sales patterns.

### Top Products Analysis
Found top 10 best-selling products.

### Discount Impact Analysis
Studied how discounts affect profitability.

## Key Business Insights
- Technology category generated the highest sales.  
- Consumer segment contributed the largest share of sales.  
- Higher discounts negatively impacted profitability.  
- Certain products generated significantly higher revenue than others.  
- Sales showed noticeable variation across different time periods.
